# Gator Cup

## Overview
**Gator Cup** is a soccer-style game where the player aims and shoots a ball into the goal to score points. The goal is to reach a certain number of points within 60 seconds. As the levels increase, the open areas in the goal get smaller, making the game more difficult. The player must progress through and beat all three levels to win the game.
